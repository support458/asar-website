Автоматизация блога через Google Drive и Google Docs

Публикации можно обновлять без ручной верстки: команда пишет материал в Google Docs, сайт автоматически синхронизирует текст, изображения и метаданные.

1

Автор создает материал в Google Docs по шаблону.

2

Google Sheet хранит slug, язык, категорию, SEO-поля.

3

Скрипт импорта (Drive API + Docs API) выгружает и конвертирует контент.

4

Сайт обновляет материалы, картинки и sitemap автоматически.

docs/ -> parser -> /content/{lang}/posts/*.html -> build -> sitemap.xml
